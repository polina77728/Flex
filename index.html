Flex-верстка определяет способность гибкого элемента растягиваться или сжиматься для заполнения собой доступного пространства.
Во Flex-верстке есть основной контейнер и вложенные в него элементы

Модуль CSS flexible box layout определяет модель CSS box, оптимизированную для дизайна пользовательского интерфейса, и компоновку элементов в одном измерении. В модели гибкой компоновки дочерние элементы гибкого контейнера могут располагаться в любом направлении и могут "изменять" свои размеры, либо увеличиваясь, чтобы заполнить неиспользуемое пространство, либо сокращаясь, чтобы избежать переполнения родительского. Можно легко манипулировать как горизонтальным, так и вертикальным выравниванием дочерних элементов.

В следующем примере контейнеру присвоено значение display: flex, что означает, что три дочерних элемента становятся элементами flex. Значение justify-content было установлено равным space-between для равномерного размещения элементов по главной оси. Между каждым элементом размещается одинаковое пространство, при этом левый и правый элементы располагаются на одном уровне с краями гибкого контейнера. Вы также можете видеть, что элементы растягиваются по поперечной оси из-за значения по умолчанию align-items равное stretch. Элементы растягиваются по высоте гибкого контейнера, благодаря чему каждый из них кажется таким же высоким, как самый высокий элемент.

Для управления шириной контента и промежутками между ними, растягивающимися горизонтально по главной оси, используют свойство justify-content. Для управления вертикальным поведением элементов служит свойство aling

Свойство flex-direction CSS задает способ размещения элементов flex в контейнере flex, определяя главную ось и направление (нормальное или обратное).

Принимаются следующие значения:
row
Главная ось контейнера flex определяется так, чтобы совпадать с направлением текста. Точки main-start и main-end совпадают с направлением содержимого.
row-reverse
Работает так же, как row но точки main-start и main-end противоположны направлению содержимого.
column
Основная ось контейнера flex совпадает с осью блока. Точки main-start и main-end такие же, как точки before и after в режиме записи.
column-reverse
Работает так же, как column но main-start и main-end противоположны направлению содержимого.

Свойство flex-wrap CSS определяет, будут ли элементы flex принудительно помещаться в одну строку или их можно переносить в несколько строк.
Если перенос разрешен, он задает направление, в котором будут располагаться строки.

Принимаются следующие значения:
nowrap
Элементы flex расположены в одной строке, что может привести к переполнению контейнера flex. Перекрестный запуск эквивалентен start или перед, в зависимости от flex-direction значения. Это значение по умолчанию.
wrap
Элементы flex разбиваются на несколько строк. Значение cross -start эквивалентно значению start или перед зависимым flex-direction значением, а значение cross-end противоположно указанному значению cross-start.
wrap-reverse
Ведет себя так же, как wrap но перекрестное начало и перекрестный конец изменяются местами.
